## Setup

1. Clone the repo.
2. Install dependencies: `npm install`
3. Configure `.env.local`:
   ```env
   TELEGRAM_BOT_TOKEN=your_bot_token
   TELEGRAM_CHANNEL_ID=your_channel_id
   ```
4. Run the fetch script: `npm run fetch`
5. Start the server: `npm run dev`
